# EduMedEx 🌍🩺

**Empowering Indian Medical Graduates to Practice Globally**

EduMedEx is a professional medical career consultancy platform that bridges the gap between Indian NEET PG doctors and international medical opportunities. We provide expert counseling, specialty-specific training, and verified global referrals — all in one place.

---

## 🌐 Live Website

> Open `edumedex/index.html` in your browser to view the site locally.

---

## ✨ Features

- **Free Consultation Form** — Doctors can register and request a 1-on-1 counseling session directly from the homepage
- **Animated Stats Counter** — Live count-up animation showcasing 500+ doctors, 15+ countries, and 95% satisfaction
- **Scroll Animations** — Smooth fade-up transitions on every section using IntersectionObserver
- **Active Nav Tracking** — Navbar highlights the current section as you scroll
- **Mobile Responsive** — Fully responsive with a hamburger menu and slide-in drawer for mobile
- **Working Forms** — Consultation and contact forms with success feedback states
- **Sticky Glassmorphism Navbar** — Blur backdrop navbar that reacts to scroll

- **HTML5** — Semantic, accessible markup

- No frameworks, no build tools — runs directly in any browser
